# Websocket battleship server

## Installation

1. Install:
   - server/package.json `npm install`
2. Start:
   - server/package.json `npm run start`

## Play

- `cd client`
- `npm run start`

  Open link http://localhost:8181

**All commands**

1. Start

```bash
 npm run start
```

2. ESLinter

```bash
 npm run lint
```

3. Prettier

```bash
 npm run format
```
